Linux

A version of DCSS may be available in your distribution's package repository (look for a package 'crawl' or similar), but beware! The version could be pretty old. Remember, you can always build from source.

Ubuntu, Debian & other Debian-derivatives

DCSS packages are available for i386, amd64 and armel architectures. Follow these instructions:

echo 'deb http://crawl.develz.org/debian crawl 0.15' | sudo tee /etc/apt/sources.list # Install the source repository
wget http://crawl.develz.org/debian/pubkey -O - | sudo apt-key add - # Install the DCSS signing key
sudo apt-get update # update your package list
sudo apt-get install crawl # install console version
sudo apt-get install crawl-tiles # install tiles version

Unofficial packages for Fedora 21/20

Player BlasterBlade has provided Fedora packages for Crawl (tiles and console builds) in the openSUSE Build System repository. To install the tiles version, run the following (for Fedora 20, replace the 21 in the URL):

cd /etc/yum.repos.d/
sudo wget http://download.opensuse.org/repositories/home:nazar554/Fedora_21/home:nazar554.repo # Install the source repository
sudo yum install crawl-sdl crawl-data # install tiles version
sudo yum install crawl-console crawl-data # install console version

Source Code

If you'd like to compile DCSS yourself (or make changes...), you can download the git repository here: https://gitorious.org/crawl/crawl/source/HEAD. For help using git, check out the quickstart guide. For help compiling DCSS, check INSTALL.txt.
